# 🌸 Flowee

**Setiap Bunga, Cerita yang Mengalir**

Landing page untuk Flowee - studio bunga artisan yang menghadirkan rangkaian bunga segar dan elegan untuk setiap momen spesial Anda.

## 🚀 Fitur

- ✨ Design system "Tropical Luxury Editorial"
- 📱 Fully responsive (mobile-first)
- 🎨 Organic shapes & batik-inspired patterns
- ⚡ Fast loading (pure HTML/CSS/JS, no frameworks)
- 🌿 6 kategori produk bunga
- 💬 Integrasi WhatsApp langsung
- 📋 FAQ accordion
- 🎯 Smooth scroll & fade-in animations

## 🛠️ Tech Stack

- Pure HTML5
- CSS3 (Custom Properties, Grid, Flexbox)
- Vanilla JavaScript
- Google Fonts (Playfair Display, DM Sans, Cormorant Garamond)
- Material Icons

## 📦 Deployment

### Deploy ke Vercel

```bash
# Install Vercel CLI (jika belum)
npm i -g vercel

# Login ke Vercel
vercel login

# Deploy
vercel --prod
```

### Local Development

```bash
# Jalankan local server
python3 -m http.server 8000

# Atau gunakan Live Server extension di VS Code
```

Buka browser: `http://localhost:8000`
